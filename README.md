# Estrutura de Dados

Este repositório contém implementações e exercícios resolvidos sobre **estruturas de dados**.  
O objetivo é consolidar o aprendizado prático e teórico, organizando exemplos de código e testes.

---

## 📂 Organização do Repositório

- `docs/` → Resumos teóricos, anotações e diagramas.
- `src/` → Implementações de estruturas de dados.

---

## 🚀 Estruturas Implementadas

- Pilhas
- Filas (simples e com prioridade)

---

## 🛠️ Tecnologias Utilizadas

- Linguagem: **C++**
- Compilador: `gcc`
- Sistema de testes: funções próprias em `testes/`

---

- Links úteis
- Diagramas explicativos
